# mist_org_rftemplate (Resource)

This manages the RF Templates.The RF Templates can be used to define Wireless Access Points radio configuration, and can be assigned to the sites


## Example Usage

```terraform
resource "mist_org_rftemplate" "rftempalte_one" {
  name         = "rftempalte_one"
  org_id       = mist_org.terraform_test.id
  band_24_usage = "auto"
  band_5 = {
    ant_gain = 2
    power    = 8
    channels = [
      60,
      104,
      132
    ]
    bandwidth = 20
  }
  band_6 = {
    ant_gain = 2
    power    = 8
  }
  band_24 = {
    ant_gain          = 1
    allow_rrm_disable = true
    power_min         = 18
    power_max         = 18
    bandwidth         = 20
  }
  ant_gain_5   = 2
  ant_gain_6   = 2
  ant_gain_24  = 1
  country_code = "FR"
}
```

## Import
Using terraform import, import `mist_org_rftemplate` using the `import` command:
```shell
# Gateway cluster can be imported by specifying the org_id and the rftemplate_id
terraform import mist_org_rftemplate.rftemplate_one 17b46405-3a6d-4715-8bb4-6bb6d06f316a.d3c42998-9012-4859-9743-6b9bee475309
```


In Terraform v1.5.0 and later, use an import block to import `mist_org_rftemplate` with `id={org_id}.{rftemplate_id}`:

```tf
import {
  to = mist_org_rftemplate.rftemplate_one
  id = "17b46405-3a6d-4715-8bb4-6bb6d06f316a.d3c42998-9012-4859-9743-6b9bee475309"
}
```
